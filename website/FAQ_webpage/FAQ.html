What to do if your PIN and server settings are not setup?

Text on screen

You dont have your PIN and server settings setup.

Screenshot

Possible cause

PIN and server settings are not set by the user.

Solution

Go to settings and set PIN and server settings on the Settings page.

What to do when home page displays that unable to fetch any survey details?

Text on screen

Unable to fetch data. Please check Internet connectivity.

Screenshot

Possible cause

The application is not able to make a request to the server.

Solution

Check internet connectivity. If Internet connectivity looks fine and the error persists, double check whether correct server details have been filled in the settings page. If you are connected to ASU WiFi network, there are chances that this error is thrown.

For mobile app, please connect to your cell data and recheck. If running through web browser, then please download TOR browser from the link below and open the app in Tor browser and check

What to do when the loader on submit screen does not go away for a long time?

Text on screen

Possible cause

The call to server is not getting completed. Possible cause is that the request was made and then the user lost internet connection.

Solution

Quit app and try again. Ensure that the internet connectivity is there and is proper, not intermittent.

What needs to be done if a blank page is displayed after the survey is started?

Text on screen

Start Survey takes the user to a blank page that has something like :{{question}}{{answer.answerText}}

Screenshot

Possible cause

The user has landed on the survey page but the app doesn’t have the questions. This should never happen, but if it happens, the cause is that the request to “get_survey” didn’t complete properly.

Solution

Quit app and try again. Ensure that the internet connectivity is there and is proper, not intermittent.
